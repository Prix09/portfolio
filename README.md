# Priya Mishra — Portfolio

A modern, full-stack developer portfolio showcasing AI/ML and Full-Stack projects.

🌍 **Live Website:** [https://portfolio-priya-mishra.vercel.app/](https://portfolio-priya-mishra.vercel.app/)

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: Radix UI
- **Deployment**: Vercel

## Projects

- **Vision Guide** — Real-time assistive AI system using YOLO
- **Hospital Management System** — Full-stack healthcare platform with JWT auth and WebRTC
- **Enterprise GenAI Knowledge Assistant** — RAG-based enterprise knowledge base
- **FixForge** — Industrial product catalog with advanced filtering
- **Travel Advisor** — AI-powered itinerary and chatbot application
- **Diabetes Prediction System** — ML regression model with Optuna optimization

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
